# Local Home Assistant Branding

This folder mirrors the Home Assistant brands repository path for this integration:

- `custom_integrations/hacomposablealarmclock/icon.svg`
- `custom_integrations/hacomposablealarmclock/logo.svg`

For local Home Assistant testing, copy the `brands` folder to your HA config directory so the final path is:

- `<ha_config>/brands/custom_integrations/hacomposablealarmclock/`

If your Home Assistant instance requires PNG assets for rendering, export these SVG files to:

- `icon.png` (256x256)
- `logo.png` (256x256)

and place them in the same directory.
